  • 强大的 shell

oh-my-zsh ​

  • zsh 配置管理器
  • 主题定制
  • 插件生态

开发相关 ​

fzf ​

  • 模糊搜索
  • 文件查找

tldr ​

  • 简洁的命令帮助

htop ​

  • 交互式进程查看器

ncdu ​

  • 磁盘空间分析

  • 快速定位大文件

httpie ​

  • 现代 HTTP 客户端

mtr ​

  • 丢包分析

  • 终端复用器

  • 窗口分屏

ranger ​

  • 命令行文件管理器
  • vim 风格操作
  • 预览支持
