前端开发概述 ​

什么是前端开发？ ​

前端开发是创建网站或应用程序的用户界面和用户交互部分的过程。前端开发者使用HTML、CSS和JavaScript等技术，确保用户能够看到并与之交互的内容正确显示和运行。

核心技术 ​

1. HTML ​

  • 语义化标签
  • 可访问性
  • SEO优化

2. CSS ​

  • 布局技术（Flexbox、Grid）
  • 响应式设计
  • CSS预处理器

3. JavaScript ​

  • ES6+特性
  • DOM操作
  • 异步编程

常用框架和工具 ​

框架 ​

  • Vue.js
  • React
  • Angular

构建工具 ​

  • Webpack
  • Vite
  • Rollup

其他工具 ​

  • Git版本控制
  • Chrome DevTools
  • VS Code

JavaScript基础知识 ​

变量和数据类型 ​

变量声明 ​

javascript
let name = 'John';
const age = 30;
var oldWay = 'not recommended';

基本数据类型 ​

  • Number
  • String
  • Boolean
  • null
  • undefined
  • Symbol
  • BigInt

引用数据类型 ​

  • Object
  • Array
  • Function

函数 ​

函数声明 ​

javascript
function sayHello(name) {
    return `Hello, ${name}!`;
}

箭头函数 ​

javascript
const sayHello = (name) => `Hello, ${name}!`;

常用数组方法 ​

  • map()
  • filter()
  • reduce()
  • forEach()
  • find()

对象 ​

对象字面量 ​

javascript
const person = {
    name: 'John',
    age: 30,
    sayHello() {
        console.log('Hello!');
    }
};

类 ​

javascript
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    sayHello() {
        console.log(`Hello, I'm ${this.name}`);
    }
}

异步编程 ​

Promise ​

javascript
const promise = new Promise((resolve, reject) => {
    // 异步操作

Async/Await ​

javascript
async function getData() {
    try {
        const result = await fetch('api/data');
        return await result.json();
    } catch (error) {
        console.error(error);
    }
}

导出 ​

javascript
