Docker 基础使用教程 ​

Docker 是什么？ ​

Docker 是一个开源的容器化平台，用于开发、部署和运行应用程序。

安装 Docker ​

macOS ​

bash
brew install docker

Windows ​

下载 Docker Desktop 并安装

基本命令 ​

拉取镜像 ​

bash
docker pull nginx

运行容器 ​

bash
docker run -d -p 8080:80 nginx

查看运行中的容器 ​

bash
docker ps

Dockerfile 示例 ​

dockerfile
FROM python:3.9
WORKDIR /app
COPY . /app
RUN pip install -r requirements.txt
CMD ["python", "app.py"]

常用操作 ​

  • 构建镜像
  • 推送到仓库
  • 容器编排
